Yuzu Tangerine Mousse | 橘柚慕斯

11月底的时候去虹口的His Aroma Patisserie探店，尝试了三款下午茶甜品，对其中的“橘柚桔”这款慕斯念念不忘。法式慕斯的空气感确实远远超过国内大量慕斯教程和配方，但我们国家的消费者毕竟水平有限，觉得吃起来还可以，就只看价格怎么样。后来买了一大堆法式甜品工具，恰好家里还有上次做剩下的日本柚子果茸，便打算用掉。

法式慕斯并非一天可以完成，此文尝试用时间顺序展示甜品制作方法。此安排基于本人近日安排，并非唯一。

第一天

橘肉冻

  1. 6个砂糖橘削皮，把皮和部分果肉混合榨成糊，取50g，日本柚子汁取10g，糖10g，放入奶锅中加热至50℃
  2. 1.5g吉利丁泡软
  3. 5g镜面果胶和10g砂糖混匀后加入奶锅中，煮至沸腾1分钟
  4. 冷却至55℃，加入泡软的吉利丁拌匀
  5. 4个砂糖橘取出果肉
  6. 冷却至35℃，加入果肉，若颜色不佳可以用色素调色
  7. 装入φ40的圆柱模具，注意厚度约1.2cm，冷冻

橘柚皮乔孔达

  1. 41g杏仁粉，41g糖粉，41g低筋面粉过筛搅匀
  2. 61g全蛋加入上一步用蛋抽搅匀
  3. 擦入适量橘皮、柚子皮搅匀
  4. 黄油融化后加入上一步搅匀
  5. 蛋白82g，砂糖49g打发至6分，烤箱平炉预热170度
  6. 分次混合，28cm方盘垫硅胶纸，入烤箱约18min
  7. 冷却后用保鲜膜包起来冷冻保存

第二天

日本柚子慕斯

柚子奶油：

  1. 23g全蛋，13g砂糖，1g低筋粉，15g日本柚子汁隔开水加热到55℃
  2. 加入1g吉利丁搅匀，离热源
  3. 加入10g黄油搅匀
  4. 过筛，贴面保存，冷藏

柚子慕斯：

  1. 吉利丁1.8g泡软，67g白巧克力隔水融化
  2. 100g淡奶油加热至55℃，离火加吉利丁
  3. 2加入融化的白巧中拌匀
  4. 加入30g柚子奶油和40g日本柚子汁拌匀
  5. 剩余150g淡奶油打发至6成，加入上一步拌匀
  6. 注入扁圆模具震一下冷冻

杏仁达克瓦兹

  1. 46g杏仁粉，10g低粉，17g糖粉混合过筛
  2. 57g蛋清+24g白砂糖1档打发至六成
  3. 粉类分三次加入蛋白中切拌均匀
  4. 用圆嘴吊线挤法挤长条，烤箱预热200度
  5. 9-11分钟烤至纹路清晰，表面微微金黄即可

巧克力配件

  1. 代可可脂白巧克力55℃以下融化
  2. 冬季降温至34℃并稳定
  3. 制作配件

淋面

  1. 10g吉利丁泡软，白巧克力85g切碎、炼乳50g放入小盆备用
  2. 75g糖和38g日本柚子汁一起煮开，不能搅拌，只需化开无需过度加热
  3. 离火加入75g葡萄糖浆搅匀，继续加热至103℃（用针形温度计）
  4. 煮好的糖水冲入巧克力和炼乳中拌匀
  5. 吉利丁隔水融化后加入上一步，再加入100g镜面果胶
  6. 贴面冷藏一晚

第三天

淋面、装饰

  1. 取出淋面，按5:1分配白色和橙色，隔水加热至融化，加入色素并用均质机打匀至无气泡
  2. 烤盘包上保鲜膜，烤网放在上面准备淋面
  3. 调温到35℃（冬季）/30℃（夏季），完成淋面
  4. 用抹刀转移到切好的达克瓦兹上，顶部挤一点柚子奶油，放上巧克力配件，完成。
